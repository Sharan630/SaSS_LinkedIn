# Database Schema

## Overview

PostgreSQL database schema managed with Prisma ORM.

## Main Entities

- **Users** - User accounts and authentication
- **Posts** - Generated LinkedIn posts
- **Analytics** - Post performance metrics
- **CRM** - Contact and lead management
- **Calendar** - Scheduled posts and events
- **AI Prompts** - Stored prompt templates

## Schema File

See `linkedflow-backend/prisma/schema.prisma` for complete schema definition.
